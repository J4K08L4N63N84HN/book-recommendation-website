This website is used to prepare for the Data Mining Cup 2021 as part of the course Data Mining II at the University of Mannheim.


In order to develop a recommender system for a bookstore, we need to find, for each book, a number of recommendations. To capture similarities in the book covers, we built this website and will label manually.
